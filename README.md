# Homebrew Tap for Agent of Empires

This is the official Homebrew tap for [Agent of Empires](https://github.com/njbrake/agent-of-empires) (`aoe`), a terminal session manager for AI coding agents.

## Installation

```bash
brew tap njbrake/aoe
brew install aoe
```

## Updating

```bash
brew update
brew upgrade aoe
```

## Uninstalling

```bash
brew uninstall aoe
brew untap njbrake/aoe
```

## About

Agent of Empires provides a TUI dashboard to manage tmux sessions for AI coding agents like Claude Code and OpenCode. Features include:

- Visual session management with status indicators
- Session grouping and organization
- MCP server configuration
- Multi-profile support

For more information, see the [main repository](https://github.com/njbrake/agent-of-empires).
